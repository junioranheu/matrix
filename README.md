# Matrix

## Sobre o Projeto

Matrix é uma aplicação de console desenvolvida em C# e .NET 10 que simula a vida de milhares de pessoas ao longo dos anos.

Os habitantes envelhecem, trabalham, criam relacionamentos, formam famílias, têm filhos, acumulam patrimônio e eventualmente morrem. Conforme o tempo avança, novas gerações surgem e o mundo continua evoluindo.

**O objetivo do projeto não é criar uma simulação realista da sociedade, mas servir como um ambiente para explorar conceitos de engenharia de software através de um domínio relativamente simples, mas que cresce constantemente em complexidade.**

Além da simulação em si, o Matrix funciona como um laboratório para testar ideias, arquiteturas de software, otimizações de performance e fluxos de desenvolvimento assistidos por inteligência artificial.

## Objetivos

O projeto foi criado para praticar e explorar:

- C# moderno
- .NET 10
- Arquitetura de Software
- System Design
- Modelagem de Domínio
- Clean Code
- Refatoração
- Otimização de Performance
- Paralelismo e Concorrência
- Estruturas de Dados
- Testes Automatizados
- Desenvolvimento Assistido por IA

Em vez de resolver exercícios isolados, o Matrix utiliza uma simulação em constante evolução para aplicar esses conceitos de forma prática.

## Funcionalidades

Atualmente a simulação inclui:

- Nascimento de habitantes
- Envelhecimento
- Morte
- Relacionamentos
- Casamentos
- Filhos
- Famílias e parentesco
- Evolução financeira
- Felicidade
- Carreira profissional
- Migração entre países
- Histórico de eventos de vida
- Estatísticas populacionais
- Evolução do mundo ao longo dos anos

Cada habitante possui sua própria trajetória, construída por meio de probabilidades, eventos e interações com outras pessoas da simulação.
